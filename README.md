# 小明的个人名片网页

## 这是什么？

这是一个**个人名片网页**，用纯 HTML + CSS + JavaScript 写的。打开后你会看到一张简洁大方的名片，上面有你的名字、介绍和邮箱。

## 怎么看效果？

1. 双击打开 `index.html` 文件
2. 浏览器会自动显示名片页面
3. 就这么简单！

## 名片上有什么？

- **头像**：正中间一个蓝色圆形头像（目前是占位图，显示"明"字）
- **名字**：小明
- **介绍**：正在学习用 AI 开发程序
- **邮箱**：xiaoming@example.com
- **保存按钮**：点击后会下载一个包含联系方式的文本文件
- **分享按钮**：手机上可以分享给朋友；电脑上会复制到剪贴板

## 如果要改名片内容

所有改动都在 `index.html` 中：

| 要改的内容 | 位置 | 怎么改 |
|-----------|------|--------|
| 名字 | `<h1 class="name">小明</h1>` | 把"小明"换成你的名字 |
| 介绍 | `<p class="tagline">...</p>` | 把中间的文字换掉 |
| 邮箱 | `<span class="contact-text">...</span>` | 换成你的真实邮箱 |
| 头像 | `<span class="avatar-placeholder">明</span>` | 可换成 `<img src="你的头像.jpg">` |

## 三个文件各是干什么的？

| 文件名 | 作用 |
|--------|------|
| `index.html` | 网页骨架，包含名片的所有文字和结构 |
| `style.css` | 网页外观，控制颜色、布局、手机适配 |
| `script.js` | 网页交互，控制按钮点击、动画效果 |

## 在手机上怎么看？

1. 把这三个文件发给手机
2. 用手机浏览器打开 `index.html`
3. 页面会自动适配手机屏幕大小

## 技术信息

- 纯 HTML + CSS + JavaScript，不需要安装任何东西
- 配色方案：浅蓝 + 白色
- 支持电脑、平板、手机三种屏幕尺寸
- 所有代码都有中文注释，方便以后修改
